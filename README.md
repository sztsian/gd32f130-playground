# gd32f130-playground
GD32F130 playground using [LibOpenCM3](https://github.com/libopencm3/libopencm3)

# Notes about compiling
After clone this repo, init the sub module

> git submodule init

And then compile LibOpenCM3

> pushd libopencm3 ; make ; popd
